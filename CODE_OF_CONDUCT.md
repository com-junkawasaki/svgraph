# Code of Conduct

This project is intended to be a practical and respectful place to improve SVG and DrawingML conversion tooling.

## Expected behavior

- Be respectful and constructive.
- Keep feedback focused on the code, documentation, tests, examples, and user-visible behavior.
- Assume good intent, but be precise when reporting bugs or regressions.
- Help maintainers reproduce issues with minimal SVG or DrawingML fragments where possible.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic comments that make issue or review discussion harder to follow.
- Intentionally submitting misleading reports, unsafe payloads without context, or disruptive content.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; block users; or take other reasonable action to keep the project usable.

For conduct concerns, contact the maintainer listed in `pyproject.toml`. For security vulnerabilities, follow `SECURITY.md` instead of opening a public issue.
